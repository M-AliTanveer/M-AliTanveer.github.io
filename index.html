About The Game

The project involves the development of a sophisticated mechanical vehicle simulator designed to provide a highly realistic training experience for driving heavy mechanical vehicles, particularly trucks. This simulator meticulously replicates all the gear changes found in mechanical vehicles, offering users a comprehensive understanding of the intricacies involved in gear shifting.

It features tangible gears and changeable differentials, allowing drivers to experience the full range of high and low gear ratios essential for navigating various driving conditions. Additionally, the simulator incorporates advanced functionality, including the innovative use of NDisplay in Unreal Engine, enabling the seamless sharing of displays across multiple screens belonging to multiple computers, thereby enhancing the immersive training experience.

My Contributions

  • Developed the entire simulator from scratch using primarily C++, ensuring a robust and efficient codebase.
  • Created the overall manual clutch and gearing system based on realistic world scenarios, providing an authentic driving experience.
  • Implemented the engine RPM system to accurately simulate the engine's behavior under various conditions.
  • Utilized nDisplay in Unreal Engine to share the display across multiple screens belonging to multiple computers, enhancing the immersive training experience.
  • Designed the project for dedicated hardware to simulate real-world driving conditions, increasing the training effectiveness.
  • Configured input variables to be read through TCP ports, enabling real-time data processing and interaction with the simulator.
  • Integrated real simulator hardware, including a clutch and gears, to provide hands-on training for heavy vehicle driving.
  • Packaged the entire vehicular code into a plugin, which is set to be released soon on the Unreal Engine Marketplace, making it accessible for further development and use.

About The Game

Stickman, an homage to classic stickman games, is a fighting game crafted within Unreal Engine 5, drawing inspiration from titles like Tekken. It innovates multiplayer combat with a listen server architecture and EOS integration in C++, eliminating the need for dedicated servers while ensuring robust multiplayer services. The game features a dynamic combo system for fluid combat and precise synchronization of gameplay actions across clients through efficient RPC usage.

My Contributions

  • Designed a dynamic combo system tailored for engaging combat experiences
